# Решения — task-009-review

| # | Решение | Альтернативы | Обоснование | Где зафиксировано |
|---|---|---|---|---|
| 1 | Импорты внешних модулей: type-only — из подмодуля (`from ..external.registry import ExternalUserRegistryClient`), не из пакета | Импортировать из пакета (`from ..external import ...`), считая side-effects терпимыми | Пакетный `__init__.py` имеет `@lru_cache`-фабрику и тронет `Settings()`. Для type-only это пустая работа и потенциальный источник import-time ошибок. Правило выписано в `docs/08-conventions.md` (новая секция «Импорты: side-effects в пакетных `__init__.py`») | [`docs/08-conventions.md`](../../docs/08-conventions.md), [`state/DECISIONS.md`](../../state/DECISIONS.md) |
| 2 | Раздельные fixture'ы `session` (rollback) и `nested_session` (SAVEPOINT) | Унифицировать на `nested_session` везде | Раздельные передают intent: repo-тесты по дизайну не коммитят, service-тесты коммитят. Overhead `nested_session` мал, но «универсальная» fixture скрывает архитектурное различие | [`state/DECISIONS.md`](../../state/DECISIONS.md), TASK-010 Step 0 |
| 3 | Сервисы не делают `await session.rollback()` на доменных исключениях | Оставить как было (сервисный rollback) | Контекст-менеджер сессии (middleware/handler) сам откатит при пробросе исключения. Сервисный rollback избыточен, конфликтует с SAVEPOINT-тестами и создаёт `SAWarning`. Уточнено правило #3 в `docs/08-conventions.md`; точечный фикс `EventService.delete_outcome` в Step 0 TASK-010 | [`docs/08-conventions.md`](../../docs/08-conventions.md), [`state/DECISIONS.md`](../../state/DECISIONS.md), TASK-010 Step 0 |
| 4 | `UserService.touch_last_seen` остаётся с commit | Не коммитить, оставить caller'у; throttle на N минут | Консистентность с другими write-методами важнее микрооптимизации. Throttling — отдельная задача при реальной нагрузке (когда будут метрики) | [`src/shared/services/user.py`](../../src/shared/services/user.py) |
| 5 | `StubRegistry` в `tests/integration/services/conftest.py` | Переезд в общий `tests/_stubs.py` | Stub используется только UserService-тестами; переезд имеет смысл, когда появятся unit-тесты с замоканными репо. Пока их нет, центральная локация — преждевременное обобщение | [`tests/integration/services/conftest.py`](../../tests/integration/services/conftest.py), [`state/DECISIONS.md`](../../state/DECISIONS.md) |
