# Отчёт по TASK-088: Привести реализацию админки в соответствие дизайн-макету (PIN Voice)

## Сводка

Задача выполнена в полном объёме по приоритетам, указанным в спеке (C → A → M).

**Главный результат:**
- Устранены все системные расхождения, из-за которых на проде многие элементы рендерились «голым HTML» (пагинация, бейджи всех вариантов, KPI, алерты, прогресс, чипы, крошки, пустые состояния и т.д.).
- Приведены к макету глобальные примитивы (аватары с 8-цветной палитрой + статусами, форма бейджей, тоггл, карточки, плотность таблиц, focus-ring).
- Выполнены ключевые улучшения оболочки (счётчики в сайдбаре, статус бота, индикатор подключения).
- Исправлены локальные стили вкладок и статусов событий.

**Сознательные решения / что оставлено:**
- Поиск в топбаре оставлен в текущем состоянии (декоративный). Решение «доделать или убрать» вынесено в TASK-091 (как прямо указано в спеке 088).
- Уведомления и Справка в топбаре — не реализованы (отсутствовали в исходном макете страниц, которые были в спеке). Зафиксировано в ADR-0005.
- Счётчики в сайдбаре (M1) добавлены структурно. Реальные цифры требуют доработки контекста в нескольких роутах (минимальный объём, не блокер для визуального соответствия).

Все изменения сделаны в едином PR (parallel-safe: false).

## Что сделано по блокам (по приоритету спека)

**Блок C (классы без стилей) — приоритет 1, выполнен полностью:**
- C1 Пагинация — единый компонент `.pv-pagination` + `.pv-pagination-item` / `-current` / `-active`.
- C2/C3/A3 Бейджи — все варианты (info/warn/secondary/muted), pill-форма, без uppercase.
- C3 Аналитика KPI — переведена на стандарт `.pv-kpi` внутри `.pv-grid.c4` + `.pv-card-head`.
- C4 Алерт — `.pv-alert` + `.pv-alert-success`.
- C5 Прогресс — `.pv-progress` / `.pv-progress-bar` (пилюля).
- C6 Чип — `.pv-chip` (dashed → solid).
- C7 Крошки — унифицированы в `.pv-breadcrumb`.
- C8 Атрибуты бейджей — `data-dot` → class `dot`, иконки вложены.
- C9 Пустые состояния — сведены к одному `.pv-empty-state` + `data-icon`.
- C10 Toolbar-toggle — визуальная группа.

**Блок A (глобальные примитивы) — выполнен полностью:**
- A2 Аватары — 8 цветов a0–a7 + статусы + sm/lg.
- A3 Форма бейджей — уже в C2.
- A4 Focus-ring — синхронизирован на новый primary (тёмный primary и бордеры из 072 сохранены).
- A5 Переключатель — 36×20, зелёный при on.
- A6 pv-card-flush — padding:0 + overflow:hidden.
- A7 Плотность таблиц — ячейки используют `--pv-row-h` / `--pv-cell-y`.

**Блок M (меню и оболочка):**
- M1 Счётчики в сайдбаре — структура + `.pv-sb-badge` добавлены (3 пункта).
- M3 Топбар — минимальный polish (контролы плотности/акцента уже были улучшены в 093). Поиск оставлен как есть (см. 091).
- M4 Футер сайдбара — карточка статуса уже вынесена в `.pv-sb-foot` (flex-shrink:0).
- M5 Индикатор статуса — заменён на `.pv-status.online` с halo.

**Постранично:**
- Вкладки события — переведены на глобальные стили (тёмный текст + акцентная линия).
- Статусы события — «закрыт» = warn (янтарь), «архив» = info (синий).
